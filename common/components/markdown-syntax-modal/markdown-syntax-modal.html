Phrase Emphasis

*italic*   **bold**
_italic_   __bold__

Links

Inline (title is optional):

An [example](http://url.com/ "Title")

Reference-style (title is optional):

An [example][id]. Then, anywhere else in the doc, define the link:
    
[id]: http://example.com/  "Title"

Images

Inline (title is optional):

![alt text](/path/img.jpg "Title")

Sizes:

![foo](foo.jpg =100x80)    Fix both width and height
![bar](bar.jpg =100x*)     Fix width and maintain ratio
![baz](baz.jpg =80%x*)     Fill 80% of the page width and maintain ratio

Reference-style:

![alt text][id]
    
[id]: /url/to/img.jpg "Title"

Headers

Setext-style:

Header 1
========

Header 2
--------

atx-style (closing #'s is optional):

# Header 1 #
    
## Header 2 ##

###### Header 6

Lists

Ordered, without paragraphs:

1.  Foo
2.  Bar

Unordered, with paragraphs:

*   A list item.
    
    With multiple paragraphs.
*   Bar

You can nest them:

*   Abacus
    * answer
*   Bubbles
    1.  bunk
    2.  bupkis
        * BELITTLER
    3. burper
*   Cunning

Blockquotes

> Email-style angle brackets are used for blockquotes.
    
> > And, they can be nested.

> #### Headers in blockquotes
>
> * You can quote a list.
> * Etc.

Code Spans

`<code>` spans are delimited by backticks.

You can include literal backticks like `` `this` ``.

Preformatted Code Blocks

Indent every line of a code block by at least 4 spaces or 1 tab.

This is a normal paragraph.
    
    This is a preformatted code block.

Horizontal Rules

Three or more dashes or asterisks:

---
    
* * *

- - - -

Manual Line Breaks

End a line with two or more spaces:

Roses are red,
Violets are blue.

Tables

Fruit    |Color
---------|----------
Apples   |Red
Pears	 |Green
Bananas  |Yellow

Definition Lists

Term 1
: Definition 1

Term 2
: Definition 2

Enfasi di frase

*corsivo*   **grassetto**
_corsivo_   __grassetto__

Link

In linea (titolo opzionale):

Un [esempio](http://url.com/ "Titolo")

Stile citazione (titolo opzionale):

Un [esempio][id]. Poi, da qualsiasi altra parte nel documento, definisci il link:
  
[id]: http://esempio.com/  "Titolo"

Immagini

In linea (titolo ozionale):

![alt text](/persorso/img.jpg "Titolo")

Dimensioni:

![foo](foo.jpg =100x80)    Fissa larghezza e altezza
![bar](bar.jpg =100x*)     Fissa larghezza e mantieni proporzioni
![baz](baz.jpg =80%x*)     Riempi l'80% della pagina in larghezza e mantieni proporzioni

Stile citazione:

![alt text][id]
    
[id]: /url/img.jpg "Titolo"

###### Intestazione 6

Liste

Ordinata, senza paragrafi:

1.  Mela
2.  Pera

Non ordinata, con paragrafi:

*   Mela
    
    Con paragrafi multipli.
    
*   Pera

Puoi anche innestarli:

*   Pera
    * Kaiser
*   Mela
    1.  Verde 
    2.  Gialla
        * Golden
    3. Rossa
*   Banana

Blocco citazioni

> Per definire un blocco citazione usa le parentesi angolate.
    
> > Puoi anche innestarli.

> #### Intestazioni in blocchi citazione
>
> * Puoi citare una lista.
> * Ecc.

Codice in linea

`<code>` Il codice in linea si delimita con apici inversi.
    
Puoi includere gli apici inversi nel codice `` `così` ``.

Blocchi di codice preformattato

Indenta ogni linea del blocco di codice di almeno 4 spazi o un tab.

Paragrafo normale.
    
    Blocco di codice preformattato.

Filetto orizzontale

Tre o più trattini o asterischi:

---
    
* * *

- - - -

Interruzione di riga manuale

Termina la riga con due o più spazi:

Prima riga,
Seconda riga.
